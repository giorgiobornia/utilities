
Define three int variables to which you assign the values
 $ a = 5 $,
 $ b = 35 $,
 $ c = 7 $

 Verify the value of the given expressions by printing their result to terminal.

\begin{verbatim}
  a + b * c
  a > b
  (a + b) * (a < b)
  (a + b) && (a < b)
  (a + b) || (a > b)
  ((a * c) - b) || (a > b)
  ((a * c) != b) || (a > b)
  (a > b) || (a < c) || (c == b)
\end{verbatim}
